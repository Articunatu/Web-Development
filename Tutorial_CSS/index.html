All modern websites and web applications are built using three fundamental technologies: HTML, CSS and JavaScript. These are the languages of the web.

In this part, let's focus on HTML. We will learn what HTML is all about, and why you too should learn it.

What is HTML?

It's a markup language that web developers use to structure and describe the content of a webpage (it is not a programming language).

HTML consists of elements that describe different types of content: paragraphs, links, headings, images, video, etc. Web browsers understand HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDN Web Docs

It is fun to learn HTML?

There are many reasons for learning the fundamental language of the web. Here are some:

  • To be able to create beautiful websites instead of relying on tools like Worpress or Wix
  • To build web applications
  • To impress friends
  • To have fun
